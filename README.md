# Italo Argentina - App Interna

App multiplataforma (Android, iOS, Web) para gestión interna de empleados de Italo Argentina.

## 📱 Características

- ✨ **Dashboard**: KPIs y estadísticas en tiempo real
- 📝 **Novedades**: Crear y compartir noticias internas
- 📊 **Reportes**: Positivos y negativos de empleados
- 📋 **Encuestas**: Sistema de encuestas internas
- 📄 **Documentos**: Compartir y descargar archivos
- 📌 **Solicitudes**: Licencias y actividades
- 🔔 **Notificaciones**: Push notifications en tiempo real
- 🔐 **Autenticación**: JWT con refresh token automático
- 💻 **PWA**: Funciona como aplicación web instalable
- 🚀 **Multiplataforma**: Android, iOS, Web
